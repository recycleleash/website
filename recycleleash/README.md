# www.recycleleash.com
This repo holds our website on Azure using a Static Web App.

# How did we get here?
1. Install and configure Hugo - https://gohugo.io/getting-started/installing/
2. Add theme (spectral) - https://html5up.net/spectral
3. Modify exampleSite
4. Test site offline
5. Follow document published by Microsoft on how to deploy Static Web App - https://docs.microsoft.com/en-us/azure/static-web-apps/publish-hugo
